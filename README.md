# osx-home

## Installation

1. Install Xcode Command Line Tools from [here](https://developer.apple.com/downloads/index.action).
1. Install osx-home

  ```
  mkdir ~/workspace
  git clone https://github.com/mamachanko/osx-home ~/workspace/osx-home
  ~/workspace/osx-home/bin/install_home
  ```
